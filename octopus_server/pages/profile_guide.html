Steps:
  1. Look for your username or "Account" in the top-right corner
  2. Click on the dropdown arrow next to your username
  3. Select "Profile" from the dropdown menu

2Alternative: Direct URL

You can navigate directly to the profile page using the URL:

http://your-server:5000/profile

Local development example: http://127.0.0.1:5000/profile

What You'll See in Your Profile

Parameter Categories:
  • API Credentials
    ServiceNow API keys, Jira tokens, etc.
  • Integrations
    Instance URLs, default settings
  • Notifications
    Email addresses, notification preferences
  • Preferences
    Timezone, refresh intervals, UI settings
Available Actions:
Quick Start:
  1. Access Profile using the dropdown method above
  2. Click "Add Parameter" to create your first setting
  3. Choose "API Credentials" category
  4. Add your ServiceNow API key (mark as sensitive)
  5. Create tasks - plugins will automatically use YOUR settings!
Sample Data Available:

If you ran the initialization script, the "admin" user already has sample parameters set up:

  • 3 API Credentials (ServiceNow, Jira, Slack)
  • 3 Integration settings (URLs, priorities)
  • 2 Notification preferences
  • 2 User preferences (timezone, intervals)
Troubleshooting
Can't see Profile link?
  • Make sure you're logged in
  • Check Account dropdown (top-right)
  • Try direct URL: /profile
Profile page not loading?
  • Check server console for errors
  • Verify database is initialized
  • Run init_user_params.py
